C++_Primer_第三次阅读笔记

نُشر في 2021-09-09 عُدل في 2021-09-16

Compound：复合物，复合的，参和

一条声明语句由一个基本数据类型和而后的一个声明符列表组成。

引用（reference）并非对象，初始化值必须是一个对象（不能是字面值或计算结果），相当于给对象起的别名，并绑定在一起。

int i= 1024；int &r = i；引用类型和绑定的对象严格匹配

不能定义指向引用的指针。

在声明语句中，&和*组成复合类型（引用和指针）。在表达式中，又可用做运算符（取地址符和解引用符）。

初始化空指针，用字面值nullptr.

void* 指针，可以存放任意对象的地址。

引用的类型和指针的类型与其所对应的对象类型须保持一致，但也有例外。

与普通引用不同，对常量的引用不能用作修改它所绑定的对象。

用普通引用绑定表达式的结果，会出错，但是常量引用就不会。

int i = 1;

const int &r1 = i*2;

int &r4 = r1*2;

因为表达式的结果，C++编译器会用一个临时量（temporary）来保存。而引用的目的是对绑定对象赋值，对临时量赋值逻辑上是不允许的。

要想存放常量（const）的地址，就要用到指向常量的指针。
