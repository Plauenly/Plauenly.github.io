三。变换

视图变换，模型变换

线性变换：A’ = MA(M为方阵)此时就为线性变换， 因为e.g. y’ = ax+by+cz

”通常“变换：旋转（默认旋转方向为逆时针，且绕原点旋转），切片，缩放，反转。

平移（”特殊“的变换，引入齐次坐标可以变为线性变换）
齐次坐标（homogeneous coordinates）：在二维坐标中引入第三维度, 其中0（表示向量）,1（表示点）
即(x, y, w)point = (x/w, y/w)point, if w != 0.

向量具有平移不变性，即平移以后的向量与原向量相等

仿射变换：线性变换+平移变换（先线性，后平移）

如何绕一个给定点c旋转呢？
答：先平移到原点再旋转，最后再平移回去

行列式值为+1的正交矩阵，称为特殊正交矩阵，它是一个旋转矩阵。
（因此旋转矩阵的逆 == 旋转矩阵的转置）
变换矩阵的逆矩阵 就相当于 该变换的逆操作

四。视图变换

类比拍照的步骤：建模，找角度（视图与模型变换），投影

规定：相机看向模型的方向记为g轴，相机向上铅垂方向记为t轴
包括相机所在的空间中有一个默认的标准xyz轴。

约定：拍照的相机位置应该位于原点，且相机永远朝向-z轴方向看，并且向上角为y轴。
为了达到约定，需要对相机（视图变换）与模型（模型变换）进行相同的变换。
该变换矩阵可以通过 逆旋转矩阵 来求，即 求如何将标准的xyz轴旋转为相机的g轴，t轴与gXt轴的旋转矩阵，再求逆。

右手坐标系：x X y == z

透视投影（近大远小），正交投影

正交投影(理解)：

  1. 将相机摆放好（视图与模型变换）。
  2. “去掉”z轴，将模型全部压缩到xoy平面上
  3. 变换平面到[-1,1]^2

正交投影（实际操作）：
将定义好的视域移动和缩放到canonical(正则，规范，标准)的立方体上，即[-1,1]^3.

透视投影与正交投影的区别：透视投影的远平面更大近平面更小（视锥），而正交投影为相同大小（视域）

透视投影(应用最广泛的投影)的操作：

  1. 将远平面压缩为和近平面同样大小
  2. 作正交投影

挤压远平面的规定：

  1. 与平面垂直的z轴，平面对应的z轴上的值不变
  2. 平面的中心点绝对位置不变
  3. 近平面在挤压的过程不受影响

将物体“挤压”为正交投影

网络编程

Tcp协议：面向连接，可靠，全双工，字节流，流量控制的协议
在复杂的网络环境中确定一个进程，需要ip地址+端口号

字节序：
x86系统采用小端存储（低地址存低位），网络数据采用大端存储（低地址存高位）。小同（小端相同）
所以需要使用字节序转换。

使用socket编程，实现数据在不同设备，不同平台之间传输

状态迁移：
服务器端状态：LISTEN，ESTABLISHED（被动受到请求）
状态：FIN_WAIT2，TIME_WAIT（主动断开连接），CLOSE_WAIT（被动断开连接）

网络通信的常规函数：

读写函数：

netstat -apn打印本机网络信息
哪端先请求断开连接，哪端就会进入TIME_WAIT(等待2MSL时间)的阶段
使用 setsockopt()函数可以实现端口和地址复用。
netstat -apn | grep port_id，找到端口相关的进程， kill -9 pid关闭该进程进而释放端口。

IO多路复用

传入传出参数：该参数通常为指针或引用形式，需要在传入时设置值，在函数结束时其值可能发生改变。

位图：非常大的数组，数组只存放0或1的数据，数组的下标就是查找的对象。如fd_set的下标就是fd号。

select()与poll()在存在大量并发连接但却只有少量活跃连接的情况下，仍需要遍历整个fd数组，效率比较低
解决办法：使用epoll()

水平触发（LT，默认）：
当文件描述符关联的读内核缓冲区非空时（即有数据可读），就一直发出可读信号进行通知，直到读完为止
；当关联的写内核缓冲区非满（即有空间写数据），就一直发写信号进行通知。
（这就解释了为什么当fd激活而没有处理时，会进入死循环（针对~/tmp/train_inet/server.c中select的代码））
边缘触发（ET）：
当fd关联的读内核缓冲区有可读数据进入时，仅仅发出一次读信号；同样也仅仅发出一次写信号。

文件描述符(fd)

文件描述符在形式上是一个非负整数。实际上它是一个索引值，指向内核为每一个进程所维护的该进程打开文件的记录表。
当程序打开一个现有文件或者创建一个新文件时，内核向进程返回一个文件描述符。

0：标准输入
1：标准输出
2：错误输出

自己创建的文件描述符从3开始依次递增，当中间的文件描述符被删除，新增的文件描述符存放在删除点并从此处开始递增。
e.g.
0 1 2 3 4 5
0 1 2 4 5 删除3
0 1 2 3 4 5 新增的放在3

生活有感四

发表于 2022-01-14

其实我一直认为这个世界不是一个非黑即白的世界，没有人能限制住一个人的思想，那是作为人最根本的权力，除非自己去限制自己的能力，这也时常发生。

我发现自己并不是对诱惑很有抵抗力。诱惑大部分都是高刺激事物，当我习惯接受一些高刺激的东西，我会对自己缺乏关注。但是在我今后的日子里，我自己的生活完全靠自己，也必须是自己，我需要的是低刺激的事物，反而不会让我错过享受生活的机会。高刺激的事物很爽，但终究是昙花一现，不能成为我生活的主旋律。

低刺激的事物其实来的很慢，需要点时间，但却很自然。比如我现在的目标：我想扣篮，我每天坚持一定的训练量，坚持吃早饭，在能扣篮的那刹那我想是美丽的。我们如果不控制好自己对娱乐的痴迷，再好的成长条件和机会都会被浪费掉。

其实不需要过多地讨论自然和人工的，哪个更好。只要适合我们生存的主旋律，该摒弃的摒弃，该采纳的采纳。

控制住自己的欲望，和控制自己的饮食方式一样，都是不容忽视的事情。在这个精神需求成为人们必须品的时代，我们需要控制自己的摄入量，以及质量。
